# Requisits per a l'aplicació Conversor de Fitxers

Perquè aquesta aplicació funcioni correctament, necessites tenir instal·lat el següent programari al teu sistema:

## 1. LibreOffice

Aquesta aplicació depèn completament de LibreOffice per a la conversió de fitxers.

- **Instal·lació**: Assegura't de tenir una versió recent de LibreOffice instal·lada.
- **PATH del sistema**: L'executable de LibreOffice (`libreoffice` a Linux/macOS o `soffice.exe` a Windows) ha d'estar accessible al PATH del sistema. Si no ho està, l'aplicació mostrarà un error indicant que no troba l'executable.

Pots descarregar LibreOffice des del seu lloc web oficial:
[https://www.libreoffice.org/download/download/](https://www.libreoffice.org/download/download/)

## 2. Llibreries de Python

Aquesta aplicació utilitza la llibreria `ttkthemes` per millorar l'aspecte visual de la interfície.

Pots instal·lar-la mitjançant `pip`. Es recomana fer-ho en un entorn virtual.

```bash
# Crear un entorn virtual (opcional però recomanat)
python -m venv .venv
source .venv/bin/activate  # A Linux/macOS
# .\.venv\Scripts\activate  # A Windows

# Instal·lar la llibreria
pip install ttkthemes
```

El contingut d'un fitxer `requirements.txt` seria:

```
ttkthemes
```
